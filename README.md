# XML Sitemap Analyzer

A Python tool for analyzing XML sitemaps and extracting URLs for SEO auditing.

## Features

- Read XML sitemaps
- Extract all URLs
- Detect duplicate URLs
- Count total URLs
- Export results to CSV

## Installation

```bash
pip install -r requirements.txt
```

## Usage

Add your sitemap URL inside:

```text
sample_sitemap.txt
```

Run:

```bash
python sitemap_analyzer.py
```

Output:

```text
sitemap_report.csv
```

## Example

Input:

```text
https://example.com/sitemap.xml
```

Output:

- Total URLs
- Unique URLs
- Duplicate URLs
